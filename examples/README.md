# ForcePAD Examples

This directory is reserved for curated ForcePAD example models and teaching packs.

The current bundled installer samples live in `bin/release/samples/`. Future examples should be added here when they are intended for GitHub visitors, documentation, classroom use, or tutorials.

Recommended structure:

```text
examples/
  cantilever-beam/
    cantilever-beam.fp2
    README.md
    sketch.png
    displacement.png
  stress-concentration/
    stress-concentration.fp2
    README.md
    mises.png
  topology-optimisation/
    topology-optimisation.fp2
    README.md
    before.png
    after.png
```

Suggested first examples:

- Cantilever beam
- Stress concentration plate
- Load path exploration
- Topology optimisation design domain
- Image-based architectural structure

Each example should explain what to open, what to change, and what structural behaviour to observe.
