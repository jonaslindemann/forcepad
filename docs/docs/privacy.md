# Privacy Policy

Last updated: May 2026

ForcePAD is a desktop application. It does not collect, transmit, or store any personal data. No account is required to use the application.

This website (the ForcePAD documentation site) may collect standard web server access logs (IP address, browser, pages visited) for operational purposes. No tracking cookies or analytics are used beyond what the hosting platform (GitHub Pages) may collect by default.
