# Physics Mode

In physics mode you place forces and boundary constraints on the structure drawn in sketch mode. Switch to this mode using the **Physics** button in the lower right corner of the window.

Placement of both forces and constraints is a two-step click interaction: the first click sets the position, and dragging while holding the button down sets the direction.

## Creating forces

1. In physics mode, click the **force tool** in the left toolbar.
2. Click on the point in the structure where the force should act.
3. While holding the mouse button down, move the mouse to set the direction of the force.
4. Release the mouse button to confirm.

The force arrow appears at the selected position and angle. Multiple forces can be placed in the same way.

## Creating constraints

1. Click the **constraint tool** in the left toolbar.
2. Click on the point where the constraint should be applied.
3. While holding the mouse button down, move the mouse to set the constraint direction.
4. Release the mouse button to confirm.

Constraints are visualized with a support symbol at the applied position.

!!! note
    Both forces and constraints should be placed on or near the drawn structure (non-white pixels). Placing them on empty (white) areas will have no effect on the FEM calculation.
