# ForcePAD Roadmap

This roadmap communicates project direction without promising dates.

## Near Term

- Improve the Qt-based ForcePAD workflow and polish.
- Add curated example models with screenshots and teaching notes.
- Add short demo videos and README/front-page GIFs.
- Improve packaging notes for Windows, macOS, and Linux.
- Expand quick-start and student learning material.

## Medium Term

- Improve topology optimisation examples and documentation.
- Add more export and interoperability documentation.
- Clarify the C++ architecture for contributors.
- Add regression examples for common structural mechanics scenarios.
- Improve HiDPI and window resizing behaviour.

## Longer-Term Ideas

- More scriptable workflows.
- More explicit CALFEM interoperability examples.
- Plugin or extension points for teaching experiments.
- Additional meshing and visualization options.
- AI-assisted workflows for explaining results or generating classroom exercises.

## Community Input

Use GitHub Issues for concrete bugs and feature requests. GitHub Discussions should be enabled in the repository settings for usage questions, teaching ideas, and example requests.
